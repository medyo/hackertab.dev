## Development

Please use the develop branch. 
don't forget to rename `.env.example` to `.env`

```bash
$ git clone --branch develop git@github.com:medyo/hackertab.dev.git
$ cd hackertab.dev
$ yarn
$ yarn start
$ # Then visit http://localhost:3000
```
